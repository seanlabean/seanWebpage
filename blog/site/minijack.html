MiniJack

MiniJack is a blackjack simulator I built as an initial foray into programming in JavaScript. While it does not include any fancy animations or user inputs, it was sufficient in teaching me about JavaScript events, datatypes, and logic-flow as well as supplemental skills like designing assets. It is still very much a work in progress, but once I'm comfortable showing it off, I'll embed the project here. My ultimate vision is to have a aesthetically pleasing app where, in playing blackjack, accurate statistical information is provided to the player: the probability of success, when hitting or staying is prefered.

With 100 Rabbits' DotGrid, I made an asset sheet for the card deck. I've included links to the page and the .grid file here if you want to use either in your own projects. Of course, the card assets can be used for any sort of card-based game, so have at it!
